# 🏦 Banking SOC Monitoring Dashboard (Splunk + Python)

## 📌 Overview
This project simulates a real-world Security Operations Center (SOC) environment for a banking system.

I generated realistic network and authentication logs using Python, then ingested and analyzed them in Splunk to detect suspicious activities such as brute-force login attempts.

---

## 🚀 Key Features

- 📈 Network traffic monitoring over time  
- 🌍 Top source IP analysis  
- 🔐 Protocol distribution insights  
- 🚨 Brute-force attack detection  
- ⏱️ Failed login trends  

---

## 🧠 Use Case

This project replicates how banks and telecom companies monitor:
- Unauthorized access attempts  
- Suspicious IP behavior  
- Network traffic anomalies  

---

## 🛠️ Tech Stack

- Python (log simulation)
- Splunk (SIEM & visualization)
- CSV (data ingestion)
